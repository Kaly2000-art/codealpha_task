# 🏨 CodeAlpha Task 1 - Hotel Reservation System

## 📌 Project Overview

The **Hotel Reservation System** is a console-based Java application developed as part of the **CodeAlpha Java Programming Internship**. It allows users to manage hotel room reservations through a simple and interactive menu-driven interface.

## ✨ Features

* 🛏️ View available rooms
* ✅ Book a hotel room
* ❌ Cancel a reservation
* 📋 Display reservation details
* 💻 User-friendly console interface

## 🛠️ Technologies Used

* Java
* Object-Oriented Programming (OOP)
* VS Code

1. Compile the Java program:

   ```

   ```
2. Run the program:

   ```
   java HotelReservationSystem
   ```
