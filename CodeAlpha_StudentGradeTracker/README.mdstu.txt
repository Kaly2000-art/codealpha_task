# CodeAlpha Task - Student Grade Tracker

## Description

This project is a Java-based Student Grade Tracker developed as part of the CodeAlpha Internship. It allows users to enter student marks, calculate the total and average marks, and display the corresponding grade.

## Features

* Add student marks
* Calculate total marks
* Calculate average marks
* Display student grade
* Console-based user interface

## Technologies Used

* Java
* VS Code

## How to Run

1. Compile the program:

   ```
   javac StudentGradeTracker.java
   ```
2. Run the program:

   ```
   java StudentGradeTracker
   ```

## Sample Output

* Enter student details
* Enter marks
* Display total marks
* Display average marks
* Display final grade
